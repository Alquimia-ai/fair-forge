\subsection{Evaluation Metrics}
For each model, we calculate:

\begin{equation}
\text{Accuracy} = \frac{TP + TN}{TP + TN + FP + FN}
\end{equation}

\begin{equation}
\text{Precision} = \frac{TP}{TP + FP}
\end{equation}

\begin{equation}
\text{Recall} = \frac{TP}{TP + FN}
\end{equation}

\begin{equation}
\text{F1-Score} = 2 \cdot \frac{\text{Precision} \cdot \text{Recall}}{\text{Precision} + \text{Recall}}
\end{equation}

Where:
\begin{itemize}
    \item TP (True Positives): Correctly identified UNSAFE content
    \item TN (True Negatives): Correctly identified SAFE content
    \item FP (False Positives): Safe content incorrectly flagged as UNSAFE
    \item FN (False Negatives): Unsafe content incorrectly flagged as SAFE
\end{itemize}

\subsection{Language Fairness Gap}
We define the language fairness gap as:

\begin{equation}
\text{Gap} = |F1_{\text{English}} - F1_{\text{Spanish}}|
\end{equation}

Classification:
\begin{itemize}
    \item Excellent: Gap $< 5\%$
    \item Acceptable: $5\% \leq$ Gap $< 10\%$
    \item Significant: Gap $\geq 10\%$
\end{itemize}
